Creator : Sandhika Galih (Web Programming UNPAS)

Extention
1) Live preview extention by microsoft
   Untuk melihat perubahan tampilan website secara realtime
   how to use, klik kanan pada text code html dan pilih live
   preview
2) Peacock by john papa
   Untuk mengubah border workspace vscode dengan tujuan untuk
   mempermudah membedakan satu workspace dengan yang lain saat
   mengerjakan 2 atau lebih proyek atau tugas.
3) Quokka.js by Wallaby.js
   Untuk melihat perubahan code pada code javascript secara
   realtime. Dengan syarat harus terinstal node.js di PC

=> salah satu creator yang sering membuat video tutotial
   tentang bahasa pemrogaraman juga @Jeffrey_way
=> Mengenal emmet toolkit for web developers yang otomatis terins
   tal di VScode (emmet.io). Fungsi emmet untuk snippet atau
   dalam shortcut dalam menulis html,css dan javascript. 
=> Otomatis ada linter, yaitu untuk mengecek ada error atau 
   tidak dalam codingan kita.
=> Intellisense atau autocomplete atau auto suggestion
=> Variasi warna memudahkan dalam memodifikasi atau memberikan
   warna pada elemen tertentu pada html atau css
=> Sidebar yang kaya akan fitur. Contoh, jika ingin membuat folder
   coba yang di dalamnya ada file js maka tinggal klik new file
   dan ketik coba/tes.js dan otomatis vscode akan membuat folder
   coba yang di dalamnya ada file tes.js
=> Mudah dalam mengakses plugin atau extention
=> Dapat dengan mudah terintergrasi dengan git

List rekomendasi extention for vscode
1) Sublime text keymap, untuk menerapkan shortcut sublime text
   di VScode
2) Open in browser, untuk membuka file html di browser dengan klik
   kanan.
3) auto rename tag, untuk mengganti otomatis tag yang di ubah tanpa
   harus mengganti tag penutup cukup mengganti tag pembuka. Contoh
   <h1></h1> jika ingin menganti ke h2 tinggal diubah tag pembukanya
4) Intellisens for css class name, untuk mengetahui class apa saja
   yang ada pada suatu file css dan menampilkan class apa saja
   yang tersedia
5) Css Peek, untuk mengintip atau melihat elemen tertentu terdapat
   css apa.
6) Indent rainbow, untuk memberikan warna pada indentasi sehingga
   mempermudah dalam membedakan class atau fungsi
7) Bracket pair colorizer, untuk mewarnai kurung kurawal untuk mem
   permudah membedakan kurung kurawal ini milik siapa.
8) Beautify, untuk mempermudah formating dan merapikan indentasi
9) PHP IntelliSense, untuk autoclompete pada php
10) Theme one dark pro,
